# Login Page

> ## Regras
1. Os campos devem começar sem exibir mensagem de error ok
2. O botão de fazer login deve começar desabilitado ok
3. Após digitar algo em algum campo, a mensagem de erro só deve sumir se o campo for válido ok
4. Validar e-mail quando o usuário digitar no campo ok
5. Mostrar mensagem de erro se o e-mail for inválido ok
6. Remover mensagem de erro se o e-mail for válido ok
7. Validar senha quando o usuário digitar no campo ok
8. Mostrar mensagem de erro se a senha for inválida ok
9. Remover mensagem de erro se a senha for válida ok
10. Habilitar o botão de fazer login se todos os campos forem válidos ok
11. Desabilitar o botão de fazer login se algum campo for inválido ok
12. Exibir loading no início da ação de login ok
13. Chamar o método de autenticar ok
14. Exibir mensagem de erro caso o login falhe ok
15. Esconder loading no fim da ação de login ok
16. Fechar streams quando a página for encerrada ok
